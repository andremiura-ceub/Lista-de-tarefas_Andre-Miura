Lista de tarefas André Miura

Este é um aplicativo simples de uma lista de tarefas(To-Do List) desenvolvido com HTML, CSS e JavaScript.

Funcionalidades
- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Salvar automaticamente no **Armazenamento local** 


Como usar
1. Baixe o repositório.
2. Abra o arquivo "index.html".
3. Digite uma tarefa no campo de texto e clique em **Adicionar**.
4. Clique na tarefa para marcar como executada.
5. Clique em excluir, para excluir a tarefa selecionada.


Desenvolvido para sistematização do curso Engenharia e Projeto de Software.
